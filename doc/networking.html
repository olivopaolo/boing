Boing: networking tools

Documentation

The following tools are intended to support testing and evaluation of networking functionalities:

  • Redirect data from multiple inputs to multiple outputs.

    Inputs and outputs must be defined using URLs. A guide of the accepted URLs is available here.

    > python3 test/run.py [-i <input>] [-o <output>]
                      [-h, --help]

Options:
  -i <input>
     Define an input

  -o <output>
     Define an output

  -h, --help
     Display this help and exit

    Specific OSC usage examples.

  • Print debug details for URLs:

    > python3 test/url.py <urls>
   
Options:
  -h, --help   
     Display help and exit
