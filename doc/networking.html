The following tools are intended to support testing and evaluation of networking functionalities:

  • Redirect data from a source to an output.

    Source and output can be even a TCP/UDP socket, a file or respectively the stdin or the stdout. SLIP encode/decode is available for TCP sockets and files.

    • > python3 test/utils/netcat.py [-l <source>] [<output>]
                               [-h, --help]
   
Options:
  -l <source>                
     Listen from the specified source

  -h, --help   
     Display help and exit

  • Create a TCP echo server:

    > python3 test/tcp/echo.py [options]
   
Options:
  -o, --host= <host>
     Set server host

  -p, --port= <port>
     Set server port

  -l <logging-level>
     Set logging level

  -h, --help   
     Display help and exit

  • Print urls's debug details:

    > python3 test/urldebug.py <urls>
   
Options:
  -h, --help   
     Display help and exit
