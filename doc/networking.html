The following tools are intended to support testing and evaluation of networking functionalities:

  • Dump or redirect packets received at the UDP/TCP socket specified by url.

    > python3 test/utils/netcat.py [-l <logging-level>] <url> [<redirect-url>]
                               [-h, --help]
   
Options:
  -l logging-level
     Set logging level

  -h, --help   
     Display help and exit

  • Create a TCP echo server:

    > python3 test/tcp/echo.py [options]
   
Options:
  -o, --host= host
     Set server host

  -p, --port= port
     Set server port

  -l logging-level
     Set logging level

  -h, --help   
     Display help and exit

  • Print urls's debug details:

    > python3 test/urldebug.py <urls>
   
Options:
  -h, --help   
     Display help and exit
