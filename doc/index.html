Boing

Boing is a toolkit for supporting the research on the use of gestures in the field of Human-Computer Interaction (HCI).

Authors: Nicolas Roussel & Paolo Olivo.

Boing is freely available under the GNU General Public License version 2 (local version).

Dependencies

Boing requires Python version 3.2 or newer and PyQt 4.

Specific notes are available for the following platforms:

  • Ubuntu 11.04 (Natty Narwhal)
  • Windows 7 (Service Pack 1)
