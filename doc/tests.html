Boing: tests

Run all available tests:

> cd <BOING-DIRECTORY>
> python3 setup.py test

Run test for a specific boing module:

> python3 boing/test/run.py [-h] [MODULE [MODULE ...]]

If no module is specified, all modules are tested; otherwise only the
specified modules will be executed.

Available modules:
 dns_sd, core, json, osc, slip, tcp, udp, utils

Positional arguments:
  MODULE      module to be tested

Optional arguments:
  -h, --help  show this help message and exit
