Boing: DNS-SD tools

Documentation

General documentation on DNS-SD can be found at http://www.dns-sd.org/.

The following tools are intended to support testing and evaluation of dns-sd service-based functionalities:

  • Announce a service using DNS-SD: 
    > python3 test/dns_sd/announce.py [options] <name> <type> <port>
   
Options:
  -o, --host= host           
     Set service host name

  -d, --domain= domain         
     Set service domain name

  -t, --txtrec= record         
     Set txt record value 
     (e.g. "{'value':'1', 'name':'example'}")

  -h, --help   
     Display help and exit

  • Browse available DNS-SD services: 
    > python3 test/dns_sd/browse_and_resolve.py [options] <type>

Options:
  -d, --domain= domain         
     Set service domain name

  -h, --help
     Display help and exit
