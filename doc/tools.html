Boing: tools

Documentation

The following tools are intended to support the development, testing and evaluation of multi-touch enabled projects:

  • Redirect data from multiple inputs to multiple outputs.

    Inputs and outputs must be defined using URLs.

    > python3 test/run.py [-i <input>] [-o <output>]
                      [-h, --help]

Options:
  -i <input>
     Define an input

  -o <output>
     Define an output

  -h, --help
     Display this help and exit

    Detailed utilities, OSC, TUIO and functions usage examples.

  • Print debug details for URLs:

    > python3 src/boing/utils/url.py <urls>
   
Options:
  -h, --help   
     Display help and exit
  • DNS-SD tools
  • display device tools
