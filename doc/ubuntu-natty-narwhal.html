Boing on Ubuntu 11.04

Python

Boing needs Python version 3.2 or newer. To install, type: 

$ sudo apt-get install python3

Qt 4

In order to install Qt libraries, type:

$ sudo apt-get install libqt4-dev

PyQt

Prior to compile PyQt, you will first need to compile and install SIP. Boing has been tested with SIP 4.12.2 and PyQt 4.8.4, which it is also available from Riverbank.

In order to compile SIP, type:

 $ sudo mv sip-4.12.4.tar.gz /opt/
 $ cd /opt/
 $ sudo tar -zxvf sip-4.12.4.tar.gz
 $ cd sip-4.12.4/
 $ sudo python3 configure.py
 $ sudo make 
 $ sudo make install

The same procedure is applied for compiling PyQt:

 $ sudo mv PyQt-x11-gpl-4.8.5.tar.gz /opt/
 $ cd /opt/
 $ sudo tar -zxvf PyQt-x11-gpl-4.8.5.tar.gz
 $ cd PyQt-x11-gpl-4.8.5/
 $ sudo python3 configure.py
 $ sudo make 
 $ sudo make install

DNS-SD services

Eclectic's dns_sd code uses the libdns_sd library, which in Ubuntu is provided by the Avahi Apple Bonjour compatibility library. To get it, type the following in a terminal:

sudo apt-get install libavahi-compat-libdnssd-dev

In order to use the dns_sd functionalities provided by such library, the Avahi daemon must be running.

sudo apt-get install avahi-daemon sudo avahi-daemon

WARNING! Avahi services cannot be found on the loopback interface; for this reason when working offline, it is necessary to set the IP address of the Ethernet or the Wireless interface to a dummy address. To achive this, for example type in a terminal: 

  sudo ifconfig eth0 192.168.0.1
