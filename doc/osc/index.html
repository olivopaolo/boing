Boing: OSC tools

Documentation

General documentation on OSC can be found at http://opensoundcontrol.org/

  • Record an OSC stream received at an UDP/TCP socket or send a recorded stream to a target UDP/TCP socket.

    • > python3 test/osc/recplay.py record <host> <file>
                              play|loop <file> <host>
                              [-h, --help]
   
Options:
  -h, --help   
     Display help and exit

  • Dump to stdout any OSC packet received from an UDP/TCP socket or read from an OSC log file:

    • > python3 test/osc/oscdump.py <url>
                              [-h, --help]
   
Options:
  -h, --help   
     Display help and exit
