Boing: TCP tools

Documentation

General documentation on OSC can be found at http://opensoundcontrol.org/

The following tools are intended to support testing and evaluation of networking functionalities:

  • Dump OSC packets received at the UDP/TCP socket specified by url: 
    > python3 test/osc/endpoint.py [-l <logging-level>] <url>
                               [-h, --help]
   
Options:
  -l logging-level
     Set logging level

  -h, --help   
     Display help and exit
