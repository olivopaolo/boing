Boing: OSC examples

General documentation on OSC can be found at http://opensoundcontrol.org/

Examples:

Print to console decoded OSC received on a UDP socket:

boing -i osc://:7777

Record to a file an OSC stream received on a TCP socket1:

boing -i osc.tcp://:7777 -o log:///tmp/log

Replay an OSC log to an UDP socket:

boing -i play:///tmp/log -o osc://[::1]:7777

Replay an OSC log (at double speed and loop enabled) to a TCP socket:

boing -i "osc:///tmp/log?speed=2&loop" -o osc.tcp://[::1]:7777

1SLIP encoding is added by default for OSC packages written or read on TCP sockets or files. Use the URI attribute 'noslip' to avoid default behaviour.
