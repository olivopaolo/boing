Boing: OSC examples

General documentation on OSC can be found at http://opensoundcontrol.org/

Examples:

Record to a file an OSC stream received on a UDP socket:

> python3 test/run.py -i osc://:7777 -o osc:///tmp/log.osc.bz2

Record to a file an OSC stream received on a TCP socket1:

> python3 test/run.py -i osc.slip.tcp://:7777 -o osc:///tmp/log.osc.bz2

Replay an OSC log to an UDP socket:

> python3 test/run.py -i osc:///tmp/log.osc.bz2 -o udp://[::1]:7777

Replay an OSC log (at double speed and loop enabled) to a TCP socket:

> python3 test/run.py -i "osc:///tmp/log.osc.bz2?speed=2&loop" -o slip.tcp://[::1]:7777

Print the raw content of an OSC log:

> python3 test/run.py -i osc.slip.file:///tmp/log.osc.bz2

1OSC packets read/written on TCP sockets or files are supposed to be SLIP encoded/decoded.
