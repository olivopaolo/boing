Boing on Ubuntu 12.04

Ubuntu packages

Boing is developed in Python 3.2 and it requires the PyQt bindings to work. It is possible to get both of them from the standard repositories by typing in a terminal: 

$ sudo apt-get install python3-pyqt4 python3-setuptools

Boing

In order to complete the installation, open a terminal and type:

 $ cd <BOING-DIRECTORY>
 $ sudo python3 setup.py install
