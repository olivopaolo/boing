Boing: Developer's guide

Documentation

Developers can easily deploy Boing networks by defining simple Python scripts. The function boing.create(uri) can be used to instantiate Boing nodes. The following code can be used as an example for instantiating a simple Boing network.

#!/usr/bin/env python3
import sys
import PyQt4
import boing

# Init application
app = PyQt4.QtGui.QApplication(sys.argv)

# Create nodes
n1 = boing.create("in.tuio:")
n2 = boing.create("viz:")

# Link nodes
n2.subscribeTo(n1)

# Run
sys.exit(app.exec_())
