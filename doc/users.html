Boing: User guide

Documentation

The following tools are intended to support the development, testing and evaluation of multi-touch enabled projects:

  • Redirect data streams from multiple inputs to multiple outputs.

    Inputs and outputs must be defined using URIs1.

    > boing [-h] [-i INPUT [INPUT ...]] [-o OUTPUT [OUTPUT ...]]
        [-C [HOST:PORT]]

 optional arguments:
  -h, --help            
     Show this help message and exit

  -i INPUT [INPUT ...]  
     Define the inputs

  -o OUTPUT [OUTPUT ...]
     Define the outputs

  -C [HOST:PORT]        
     Activate console

    Detailed usage examples:

    • Generic utilities
    • OSC
    • TUIO
    • Multi-touch event processing
    • filtering

  • Print debug details for URLs:

    > python3 boing/utils/url.py <urls>
   
Options:
  -h, --help   
     Display help and exit

  • DNS-SD tools

1URI's in. and out. prefixes can be omitted since it is obvious from the argument option.
