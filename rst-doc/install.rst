
Install instructions
====================

Boing requires:

* `Python 3.2`_
* `PyQt 4`_
* numpy_
* pyparsing_

Specific notes are available for the following platforms:

.. toctree::
   :maxdepth: 1

   Ubuntu <ubuntu-precise>
   OS X 10.7 <osx-lion>
   Windows 7 <windows>

..
   It also uses the following packages in few places, that you might
   need to install:

.. _`Python 3.2`: http://python.org/download
.. _`PyQt 4`: http://www.riverbankcomputing.co.uk/software/pyqt/download
.. _numpy: http://numpy.scipy.org
.. _pyparsing: http://pyparsing.wikispaces.com
